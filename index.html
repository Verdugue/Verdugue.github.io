Mes projets

Phocéenne Auto

Application web de gestion de concession automobile permettant de gérer les véhicules, clients, fournisseurs et transactions. Développée avec PHP et MySQL.

Boutique de Poisson

Application web e-commerce avec gestion des produits, panier et paiement. Architecture Node.js (backend) et HTML/CSS/JS (frontend). Base de données incluse, lancement rapide via npm.

PokéFight

Plateforme web de combat Pokémon en temps réel. Authentification, matchmaking, progression, combats interactifs et classement. Stack moderne : React, Node.js, MySQL, Socket.io.

Challenge 48H

Application Streamlit pour explorer, entraîner et prédire les prix de maisons à partir du dataset Kaggle "House Prices". Visualisation, feature engineering, et comparaison de modèles (Random Forest, XGBoost, LightGBM).
